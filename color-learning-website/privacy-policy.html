Privacy & Data Policy

This site is a student coursework project about learning colour. This page explains, in plain language, what data exists in the project and how it is used.

Accounts and profiles (in your browser)

  • Demo “log in” uses accounts stored in localStorage on your device (username, password for local verification, display name, avatar colour, learning stats). It is not a cloud login system.
  • If you use “Log out” or another browser, that device will not share the same stored data unless you copy it yourself.

Preferences (in your browser)

  • Settings such as sound effects, interface language, colour mode, and font size are saved in localStorage so the site can remember your choices.

Community page (optional cloud sync)

  • If the maintainer enables Supabase in js/supabase-config.js, posts you publish can sync there: text you write, tag, chosen colours / palette, optional image (may be stored as a data URL), author name, likes, timestamps, and related activity or report records as designed for the assignment demo.
  • The coursework SQL schema is intentionally open for demonstration; treat shared posts as public and do not submit sensitive personal information.
  • If Supabase is not configured, community data stays only in your browser’s localStorage pattern used elsewhere in the project.

No extra analytics

This simple static project does not embed third-party advertising or analytics trackers beyond what your browser normally sends when you request pages, and what you choose to submit in forms.

账号与个人资料（仅在您的浏览器中）

  • 演示用「登录」把账号信息存放在本机 localStorage（用户名、用于本地校验的密码、显示名、头像色、学习统计等），并非云端统一账号体系。
  • 退出登录或使用另一台设备/浏览器时，不会自动与您本机的数据同步，除非您自行迁移数据。

偏好设置（仅在您的浏览器中）

  • 音效、界面语言、颜色模式、字体大小等设置保存在 localStorage，以便下次访问时还原您的选择。

社区（可选云端同步）

  • 若维护者在 js/supabase-config.js 中配置了 Supabase，您发布的帖子可能同步至该服务，一般包括您填写的文字、标签、所选颜色/色板、可选图片（可能以 data URL 形式存储）、作者名、点赞、时间戳，以及作业演示所设计的活动或举报等相关记录。
  • 课程演示用的数据库权限较为宽松；请把公开发布内容视为公开信息，不要提交真实敏感个人信息。
  • 若未配置 Supabase，社区数据仍按项目中其他页所使用的方式主要保留在浏览器本地。

分析与广告

本项目为静态作业站点，未额外嵌入第三方广告或统计脚本；除浏览器访问网页时的常规请求外，只有您在表单中主动填写并提交的内容会作为数据被保存（与上节说明一致）。
