Project Introduction

Short notice for the Color Learning coursework site.

Color Learning Website is an interactive multimedia learning website designed for the EBU6305 Interactive Media Design and Production coursework.

This project focuses on digital colour learning and introduces key concepts such as colour basics, colour encoding, colour models, colour spaces, colour gamut, gamma correction, HDR, colour management workflows, and accessibility-related colour usage.

The website is designed as a stand-alone web application using HTML, CSS, and JavaScript. It does not require a server or a real database. Instead, it uses interactive pages, visual explanations, quizzes, mini games, and community-style elements to create an engaging learning experience.

Our goal is to help users learn colour concepts through visuals, games, quizzes, and challenges. The website includes five main sections: Homepage, Learning, Game, Test, and Community. Each section supports a different part of the learning journey, from understanding concepts to practising, testing, and sharing progress.

The design also considers accessibility, multilingual support, responsive layout, ethical data use, and user-friendly interaction. Features such as high contrast mode, bilingual content, and local progress handling are included to make the website more inclusive and practical for different users and devices.

项目介绍

Color Learning Website 是一个为 EBU6305 Interactive Media Design and Production 课程作业设计的互动多媒体学习网站。

本项目聚焦于数字色彩学习，内容涵盖色彩基础、色彩编码、色彩模型、色彩空间、色域、伽马校正、HDR、色彩管理流程以及与无障碍设计相关的色彩使用等主题。

该网站使用 HTML、CSS 和 JavaScript 构建，是一个可独立运行的网页应用，不需要服务器，也不使用真实数据库。网站通过互动页面、可视化解释、测验、小游戏和社区化元素，为用户提供更有趣、更直观的学习体验。

我们的目标是帮助用户通过可视化、游戏、测验与挑战学习色彩知识。网站包含五个主要部分：首页、学习、游戏、测试和社区。每个部分都对应不同的学习阶段，从概念理解，到互动练习，再到测试反馈和学习进度分享。

在设计过程中，我们也考虑了无障碍体验、多语言支持、响应式布局、数据伦理和用户友好交互。网站加入了高对比度模式、中英双语内容和本地进度处理等功能，以适应不同用户和不同设备的使用需求。
